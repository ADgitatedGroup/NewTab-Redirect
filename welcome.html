Set Options View Changes

New Tab Redirect is an extension for Google Chrome which will allow you to open any URL when creating a new tab. That's pretty cool for pages on the internet. But... you can also specify a local file like a custom *.html file or a text file!

If you'd like an introduction, use the navigation buttons to move forward and backward. This works just like the original new tab page.

If you want to skip the intro and set your options, that's cool. You can press CTRL+T or click the new tab button at the top of the browser. Or, you can click here to access the options page.

FAQ

How do I access this extension?

In the top-right of your Google Chrome browser, click the Wrench icon, then select Tools -> Extensions. A new tab will appear. Alternatively, you can type chrome://extensions/ into the address bar.

Scroll down to New Tab Redirect! and click Options.

How do I uninstall this extension?

Follow the directions above, but choose Uninstall.
